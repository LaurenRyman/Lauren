# This repository collects a few Python programs demonstrating file manipulation, 
# game utilities, and calculation routines. Each subfolder contains its own main 
# script(s), as well as test screenshots and a local README (if applicable).
# 
# #--------------------------------------------------------------------------------
# #  Contents
# #--------------------------------------------------------------------------------
# 
# 1. File Reader
#    - Folder: File Reader/
#    - Primary Script: FileFunction2.py
#    - Other Scripts/Tests:
#      * file1.py (example or supplementary code)
#      * Filefunctiontest.png, file1test.png (screenshots / test output)
#      * README.txt (additional documentation)
#    - Purpose:
#      * Searches for a numbers.txt file anywhere on your system.
#      * If found, it changes the working directory to the file’s location, 
#        then reads and processes integer values, computing totals and averages.
# 
# 2. File Merger
#    - Folder: FileMerge/
#    - Primary Script: FileMerger.py
#    - Other Files:
#      * Multiple FilemergertestX.png images (test run screenshots)
#      * README.txt (additional documentation)
#    - Purpose:
#      * Reads the first n non-empty lines from two user-specified input files 
#        and writes them side by side into a new output file.
#      * Handles missing files, short files, and attempts a system search if files 
#        aren’t found in the current directory.
# 
# 3. Magic the Gathering Life Counter
#    - Folder: Magic the Gathering Life Counter/
#    - Primary Script: LifeCounterMagic.py
#    - Other Files:
#      * README.txt (additional documentation)
#    - Purpose:
#      * Simulates a 2+ player life counter for Magic: The Gathering.
#      * Allows adding/subtracting life totals, displaying them, and exiting.
#      * Utilizes error handling for invalid inputs and supports repeated gameplay sessions.
# 
# 4. Paint Calculator
#    - Folder: Paint Calculator/
#    - Primary Script: PaintCalc.py
#    - Screenshots/Tests:
#      * PaintCalcTest1.png, paintcalctest2largewall.png, paintcalctest3expensivepaint.png, etc.
#      * README.txt (additional documentation)
#    - Purpose:
#      * Prompts the user for wall space (in square feet) and paint price per gallon.
#      * Calculates the number of gallons needed, labor hours, and total cost.
#      * Outputs a nicely formatted table, using the tabulate library.
# 
# #--------------------------------------------------------------------------------
# #  How to Use
# #--------------------------------------------------------------------------------
# 
# 1. Clone or Download
#    - git clone https://github.com/LaurenRyman/Lauren.git
#    - Or download the repo as a ZIP and extract.
# 
# 2. Navigate to a Project Folder
#    - For example, cd "File Reader" then run:
#      python FileFunction2.py
# 
# 3. Install Dependencies (where needed):
#    - pip install pyinputplus tabulate
# 
# 4. Run the Script
#    - python <script_name.py>
# 
# 5. Check the Output
#    - Some programs produce text-based results in the terminal or create text files.
#    - Screenshots (.png) show sample runs.
